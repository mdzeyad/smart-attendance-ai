# 🧠 AI Smart Attendance System (Face Recognition)

## 🚀 Project Overview
This project is a real-time face recognition-based attendance system built using Python and OpenCV. It detects and recognizes faces from a trained dataset and marks attendance automatically.

---

## 🎯 Problem Solved
Manual attendance systems are time-consuming and prone to errors. This system automates attendance using facial recognition to improve accuracy and efficiency.

---

## 👥 Who Used It
- Educational institutions (students & teachers)
- Small organizations
- Demo AI/ML projects for learning

---

## ⚙️ Tech Stack
- Python
- OpenCV
- face_recognition library
- NumPy
